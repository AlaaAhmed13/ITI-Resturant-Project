# 🍽️ Restaurant UI Project

## 📌 Overview
This is a simple restaurant website UI built as part of an ITI training program.  
The project focuses on frontend development using React, with an emphasis on building clean layouts and reusable components.

## 🎯 Features
- Home page with restaurant landing design  
- Static menu section  
- Clean and modern UI design  
- Component-based structure using React  
- Basic navigation between pages  

## 🛠️ Tech Stack
- React.js  
- HTML5  
- CSS3  
- JavaScript (ES6+)  
- Bootstrap
